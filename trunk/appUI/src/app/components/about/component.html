Features

  • Sections The app features two sections: "Home" and "History"

    The "Home" section features list of all the movies in the form of a carousel.

    The "History" section features list of all the movies that are viewed but not watched, viewed but watched partially or watched till the end in the form of a carousel. The list is sorted according to the most recent movies visited first.

  • Carousel: The carousel features an circular list of movies.

    Each card in the carousel, shows the thumbnail for the movie, title of the movie and the state of the movie (Watched/Watching).

    Clicking on a card open up the movie player in full screen mode with basic controls.

  • Navigation: The app supports carousel navigation with mouse and keyboard.

    Use the keyboard "Left" and "Right" Keys to navigate and "Enter/Return" key to select and play a movie.

    Click the arrow keys on the left or right to navigate through the carousel list.

    Use scroll to navigate the list.

  • Player: The player fetaures basic controls like: Play/Pause button, progress bar, movie time elapsed/total movie time and close button.

How To Setup/Run

Prerequisites

  • Node with npm
  • MongoDB

API Server

  • Make sure the mongo service is running.
    mongod
  • cd to [project-root]/trunk/api
  • To generate user data, run:
    node generateData.js
  • To install dependencies and start server, run:
    npm start

UI Server

  • cd to [project-root]/trunk/appUI
  • To install dependencies and start server, run:
    npm start
  • Navigate to http://localhost:4200
